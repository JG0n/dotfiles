### Keybinds
Key | Action |
|---|---|
|Meta + D | Open the Launcher |
|Meta + R| Open the Run Menu (Run Commands without opening a terminal)|
|Meta + P | Open the Powermenu |
|Meta + S | Open the Screenshot Menu |
